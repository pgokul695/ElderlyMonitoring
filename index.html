Unified AI-Enhanced Elderly Monitoring System

Revolutionary hybrid system combining sensor-based vitals tracking with AI-powered passive monitoring to ensure elderly safety and health—both at home and remotely.

Explore Features

System Overview

🫀

Vitals Monitoring

Real-time heart rate, SpO2, and temperature tracking using advanced sensors like MAX30102 and DS18B20.

📷

AI Vision Analysis

Passive camera monitoring with AI detection for facial pain, drowsiness, and fall detection using OpenCV.

☁️

Cloud Integration

Seamless data upload to Firebase or ThingSpeak with real-time dashboard and automated alerts.

📱

Smart Alerts

Instant SMS/email notifications via IFTTT or GSM module when health thresholds are breached.

🔋

Portable Design

Battery-powered wearable unit with 3.7V Li-ion battery and TP4056 charging module.

🏠

Home Integration

Dual monitoring system with portable vitals tracker and fixed room monitoring unit.
